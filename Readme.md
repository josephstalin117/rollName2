随机点名程序 
======================================================================

#### 要求

1. 所有照片必须为`jpg`格式，其余格式均不支持。
2. 将照片的文件名改为人名。


#### 使用方法

1. 将需要点名的照片放置于`images`文件夹中。
2. 执行`GetName.java`文件，会自动生成`output.txt`文件。
3. 打开`output.txt`文件，复制所有内容。
4. 打开`js`文件夹下的`random.js`。
5. 将内容复制到`var a=[]`的`[]`中。
6. 打开`index.html`运行即可。
